# ESPHome-CCU6225-Hack
Модернизация GSM сигнализации CCU6225 для подключения к Home Assistant.

Мод заключается в том, что в шину связи SIM300 и процессора устройства встраивается ESP32. Обмен данными между GSM модулем (SIM300) и процессором CCU6225 происходит через UART порты ESP32. Весь поток даных находится под контролем ESP. Это позволяет запрашивать от процессора все данные, передавать процессору управляющие пакеты, при необходимости, блокироовать исходящие СМС, использовать CCU6225 в качестве GSM шлюза для оттправки и получения любых СМС, поддерживаются русские СМС ;)
